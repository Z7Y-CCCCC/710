# Desk Orb / 账号小星盘

这是给 240x240 小屏开发板设计的账号状态摆件原型。当前浏览器版本用于预览视觉和轮播逻辑，真正方向是让 ESP32 独立联网运行。

## 当前显示内容

- 哔哩哔哩、抖音、今日头条轮播
- 每个平台显示 logo、昵称、粉丝数
- 保留星盘、专注、生长、夜间模式
- RADAR 模式下点 `✧` 手动切换下一个平台

## 数据结构

`social-data.js` 里现在只保留公开资料三件套：

- `nickname`：昵称
- `followers`：粉丝数
- `profileUrl`：公开主页地址

当前账号：

- B站：`3546867614878349`
- 抖音：`63230246827`，已配置分享主页链接
- 今日头条：`仙翁777`，已配置分享主页链接

后续接板子时，ESP32 会自己访问 `profileUrl`，解析公开页面里的昵称、头像、粉丝数。B站已配置公开粉丝数 API；抖音当前直接 HTTP 返回为空，今日头条返回需要 JavaScript 初始化的页面，所以这两个平台需要继续找公开接口或使用缓存展示。

## 独立开发板方向

固件骨架在：

```text
firmware/desk-orb-social/
```

它的目标是不依赖电脑常开：

- ESP32 自己连 WiFi
- 定时访问公开主页
- 抓昵称、头像/头像标识、粉丝数
- 缓存最近一次成功结果
- 在 240x240 屏幕上轮播
