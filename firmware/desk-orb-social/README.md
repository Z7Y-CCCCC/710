# Desk Orb Social Firmware

这个目录是“开发板独立运行”的方向：ESP32 自己连 WiFi，定时访问公开主页，缓存并显示昵称、头像标识、粉丝数。

## 当前解决范围

- 平台：哔哩哔哩、抖音、今日头条
- 字段：昵称、头像标识、粉丝数
- 展示：240x240 小屏轮播
- 不依赖电脑常开

## 说明

各平台公开页经常改版，而且可能对 ESP32 这种请求返回不同内容。固件里已经放了通用解析入口，优先解析页面里的公开 JSON/Meta 信息；如果平台拦截，屏幕会继续显示上一次缓存。
